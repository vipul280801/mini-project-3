# 🌐 JavaScript Mini Projects

This repository contains beginner-friendly web development projects built using HTML, CSS, and JavaScript. These projects use REST APIs to fetch real-time data and provide responsive user interfaces.

---

## 📂 Projects

### 🌦 Weather App

A responsive weather application that fetches real-time weather information using the OpenWeather API.

#### Features

- Search weather by city name
- Display current temperature
- Show weather condition
- Show humidity
- Show wind speed
- Responsive design
- Error handling for invalid city names

#### Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- OpenWeather API

---

### 🎬 Movie Search App

A responsive movie search application that fetches movie details using the OMDb API.

#### Features

- Search movies by name
- Display movie poster
- Show movie title
- Show release year
- Show IMDb rating
- Show genre
- Show movie description
- Error handling for invalid movie names

#### Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- OMDb API
